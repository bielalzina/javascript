The Window Object

The MOVETO() Method

moveTo(x, y) Mou una finestra: x i y són les coordenades.

Pitja el boto CREAR per obrir una nova finestra

Pitja el boto MOURE 1 per per moure-la 250px des de la seva posició actual

Pitja el boto MOURE 2 per per moure-la a les coodenades 100, 100

Pitja el boto TANCAR per tancar la finestra
