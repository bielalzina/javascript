The LOCATION Object

Conté informació sobre l'URL actual. Amb els seus mètodes podem recarregar la pàgina o canviar-la.

PROPIETATS

hash Torna la porció de l'àncora de la URL.

/js/js_es6.asp#mark_array_from

host Indica el domini i el port

hostname Mostra el domini.

href Torna l'URL completa

pathname Torna la ruta dins el domini.

port El port especificat a l'URL si n'hi ha cap

protocol El protocol de la URL

search La part de la consulta, la que va després de ?

/js/js_es6.asp?name=gabriel

METODES

assign(url) Carrega un nou document. A l'històric es trobaran tots dos, el que hi havia i el nou.

reload() Recarrega la url actual.

replace(url) Substitueix el document actual per el nou, fins i tot a l'historial. Amb back no hi podreu tornar.
