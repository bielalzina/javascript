The NAVIGATOR Object

appCodeName Torna el codi del navegador.

appName Conte el nom.

appVersion La versió

cookieEnabled Ens diu si les cookies estan habilitades o no

geolocation Torna un objecte geolocation que ens permetrà obtenir les coordenades de l’usuari. Demana permís a l’usuari.

platform per a quin sistema operatiu ha estat compilat el navegador

userAgent Torna aquesta capçalera que el navegador envia al servidor.
